杂记

发表于 2019-07-16 | 分类于 -[我在山城]

​ 上周考试结束了，是要出去转转的，但是经历并不像我所希望的那样。最后提前回来了，本来是要写一篇记录一下图片的，拖到了现在。

​ 玩对我没有什么意思，也许我能看到我没有看到过的，让我心动的。开始就没有太大心里，最后真的是下山转转了。因为天气热放弃了几个地方，最后在佛图关走去二厂的路上严重怀疑自己出现的时机不对劲。可能最后去逛逛只是去逛逛。

​ 不管怎么样，我还是溜达了一圈，贴一下路线和图片
