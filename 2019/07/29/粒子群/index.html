粒子群

发表于 2019-07-29 | 分类于 -[算法] -[数学建模]

原理

使用条件：

* 先确定算法所需的变量，比如求函数最大值时，粒子群算法变量变量为函数的自变量，同时需要指出算法变量的取值范围，比如求函数在某个区间上的最大值。
* 给出优化目标与变量的定量关系，比如求函数最大值时，需要给出这个函数的解析式。

算法执行：

  1. 生成初始种群，每个种群有指定个数的个体，每个个体有位置和速度两个属性，位置和速度分别以一个N维向量Xi和Vi描述。其中Xi在指定的上下界内取随机值，上下界即变量的取值区间，前面已给出。Vi同，一般Vi区间取[0,1]，Xi的取值取决于目标情况。

  2. 求每个个体的优化目标取值，如求函数最大值时，此步骤为算出函数最大值。将每个个体的目标取值当作这个个体的历史最优解pBest，将所有个体中最优的pBest当作群体历史最优值gBest。

    • w为惯性系数（表示个体原来的状态改变的情况将会对下一时刻的状态改变产生影响）

    • c1为个体学习系数（表示个体倾向于趋近自己的历史最优解）

      一般c1，c2 取值在[0.5，4]之间

  4. 如果迭代次数未达到要求（如指定迭代次数），回到2，否则结束迭代。并且将gBest当作最优解。

性能

迭代速度：无

全局&&局部最优：

​ 群体个数少的时候容易陷入局部最优。取多了才更可能有全局最优，说明粒子群算法的优化性能还是靠开局，如果开始的时候生成的种群中恰好有某个个体在最优解附件，那么才可能迭代出最优解。这表明群体数量的取值要越多越好，或者说取决于局部最优区间的数量。
